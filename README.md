# Site do Spotify

The Spotify Clone project aims to replicate the core functionalities and aesthetics of the original Spotify interface. 
 By leveraging the power of Bootstrap, I've ensured that this website adapts seamlessly to various screen sizes, offering users an optimal viewing experience across devices.
